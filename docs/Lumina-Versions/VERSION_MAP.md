# Lumina Version Map

Welcome to the Lumina Versioned Repository. Use this map to navigate the features and examples across the different versions of the language.

## [Lumina-Version_2-Architecture]
The current generation of Lumina, featuring distributed networking and infrastructure orchestration.
- **[Architect v2.1](Lumina-Version_2-Architecture/v2.1-Architect-Examples/)**: Self-healing infrastructure and autonomous lifecycle management.
- **[Cluster v2.0](Lumina-Version_2-Architecture/v2.0-Cluster-Examples/)**: High-availability and fleet-wide state synchronization.

## [Lumina-Version_1-Core]
The legacy generation of Lumina, focusing on the original single-node reactive engine.
- **[Genesis Classic](Lumina-Version_1-Core/v1-Classic-Examples/)**: The original reactive seed focused on local declarative logic.
- **[Historical Archives](Lumina-Version_1-Core/)**: Access documentation for versions v1.3 through v1.9.
